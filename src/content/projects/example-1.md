# Portfolio Website - Angular & Modern Web

This project showcases a clean, elegant portfolio built with the latest Angular technologies and modern web standards.

## Tech Stack

- **Framework**: Angular 20
- **Styling**: SCSS with CSS custom properties
- **Internationalization**: Custom i18n service with JSON translations
- **Markdown**: Dynamic content loading with `marked`
- **Build**: Angular CLI with optimized production builds

## Features

### Responsive Design

The site adapts seamlessly from mobile to desktop with a mobile-first approach.

### Multi-language Support

Built-in support for English and Spanish with localStorage persistence.

### Dynamic Content

Blog posts and projects are loaded from Markdown files, allowing easy content management.

### Performance

Lazy-loaded routes, optimized images, and lean CSS result in excellent Core Web Vitals scores.

## Architecture

The component architecture follows Angular best practices:
- Standalone components
- Signals for state management
- Functional routing
- Service-based data loading

## Open Source

This portfolio is built as an example of modern web development practices and is available on GitHub.
